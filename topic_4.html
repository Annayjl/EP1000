Intro to CREATE AND VIEW A WEB PAGE ON YOUR COMPUTER

When you navigate to a web page on the Internet, the browser is doing a lot of work. The browser reads all the necessary files (HTML, CSS, and JavaScript) and interprets those raw resources to paint the complex page you see.


REQUIREMENTS:

  • Text Editor
  • An Internet Browser
  • Check out W3SCHOO

CREATE AND VIEW A WEB PAGE ON YOUR COMPUTER


  1. STEP 1: OPEN YOUR TEXT EDITOR

    2. The first step is to open your text editor. It’s important to use a “raw” text editor, and not a formatted word processor.

  3. STEP 3: SAVE YOUR FILE

    4. Your web page is now ready, but currently it only exists inside of your text editor. The next step is to save the file to your computer. If you closed the text editor now without saving, your new web page would be lost!


  4. STEP 4: OPEN YOUR WEB PAGE IN YOUR BROWSER

    5. Now you’re ready to view your new page in your browser! First, open up your browser. In the top menu, click File > Open File. Navigate to the location you saved your web page. Click on your web page file and then click Open. You should see your web page!
